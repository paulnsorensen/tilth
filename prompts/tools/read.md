Read files with smart auto-sizing. **Defaults are tuned — omit `mode` unless you specifically need to force a size.** Example: `tilth_read(paths: ["src/lib.rs", "src/mcp.rs#tool_search"])` returns full small files, hash-prefixed signatures for large code, and sliced suffix reads when requested. Suffix grammar: `path#n-m`, `path#n`, `path### Heading`, `path#symbol_name`. Use `mode: "full"` to force full content and `mode: "signature"` for hash-prefixed signature lines.

Output format: each line is prefixed `<line>:<3-hex-hash>|<content>`, e.g. `42:a1b|let x = 1;`. The hash anchors the line to its current content — paste a prefixed line back into tilth_write (mode: hash) to make a drift-safe edit. Ignore the prefix when reading; resume after the `|`.
